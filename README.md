# 🚘 CAN Bus Automotive Communication Simulator

Simulation of Controller Area Network (CAN) communication between Electronic Control Units (ECUs) using ESP32 and MCP2515.

## Features

- CAN Frame Transmission
- Message Arbitration
- Interrupt Driven Communication
- Error Detection
- Multi ECU Simulation
- Proteus Simulation

## Hardware

- ESP32
- MCP2515 CAN Module

## Software

- Embedded C
- Proteus
- Arduino IDE
